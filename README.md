1. What is the difference between var, let, and const?
ans: 
var is used for traditional its a old version, re-declarable variables, 
let for block-scoped variables whose values can change, and 
const for block-scoped variables that cannot be reassigned 

2. What is the spread operator (...)?
ans: 
The spread operator (...) is like opening up an array or object and spreading out its items. You can use it to copy, combine, or expand data easily.

3.What is the difference between map(), filter(), and forEach()?
ans: 
map() – Makes a new array by changing each element.
filter() – Makes a new array with only the elements that pass a test.
forEach() – Loops through each element without creating a new array.

4. What is an arrow function?
ans: 
An arrow function is a modern, shorter way to write a function in JavaScript using => after the parameters.
It’s handy for small functions, keeps this consistent, and makes code cleaner and easier to read.

5. What are template literals?
ans: 
Template literals are a way to write strings in JavaScript that can easily include variables or expressions.
You use backticks `` instead of quotes, and ${} to insert variables or calculations.
